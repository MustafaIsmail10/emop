# emop
eMOP: A Tool for Evolution-Aware Runtime Verification

## Include eMOP plugin in Maven project

```xml
<build>
  <plugins>
    ...
    <plugin>
      <artifactId>emop-maven-plugin</artifactId>
      <groupId>edu.cornell</groupId>
      <version>1.0-SNAPSHOT</version>
    </plugin>
    ...
  </plugins>
</build>
```
